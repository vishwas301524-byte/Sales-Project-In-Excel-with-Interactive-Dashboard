📊 Store Data Analysis Project with Interactive Dashboard (Excel)
📌 Project Objective
The goal of this project is to create an interactive annual sales dashboard (2022) for Vrinda Store using Microsoft Excel.
This helps the store owner understand customer behavior and identify opportunities to increase sales in 2023 & 2024.

📂 Dataset
Source: Vrinda Store sales data (2022)
Includes:
Order details
Customer demographics (age, gender)
Sales channels
Order status
State-wise data
Product categories
❓ Key Business Questions (KPIs)
Compare sales vs orders in a single chart
Identify month with highest sales & orders
Analyze gender-wise purchasing behavior
Explore order status distribution
Find top 10 states contributing to sales
Understand relationship between age & gender (orders)
Identify top sales channels
Determine highest selling category
Calculate % of total orders delivered
Dashboard Interaction
⚙️ Process
Data Cleaning

Checked for missing values and inconsistencies
Standardized formats and data types
Data Preparation

Structured dataset for analysis
Ensured uniform naming conventions
Data Analysis

Created Pivot Tables for each KPI
Aggregated and summarized key metrics
Dashboard Creation

Combined all visuals into one interactive dashboard
Added Slicers for dynamic filtering

📈 Key Insights

🛒 Top channels: Amazon, Flipkart, Myntra
